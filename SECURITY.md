# Security Policy

## Supported Versions

This project is currently in active development.

Security fixes will be applied to the latest version of the main branch.

| Version | Supported |
|---|---|
| main | Yes |
| older versions | No |

## Reporting a Vulnerability

If you discover a security issue, please do not open a public issue.

Instead, contact the repository owner privately or create a minimal report without exposing sensitive details.

Please include:

- A clear description of the issue
- Steps to reproduce
- Affected files or components
- Possible impact
- Suggested fix, if available

## Sensitive Data

Do not commit sensitive data to this repository.

Examples:

- API keys
- Passwords
- Database dumps
- Private `.env` files
- Production credentials
- Personal user data

## Dashboard Authentication

The FastAPI dashboard uses Basic Authentication. Default credentials are intended only for local development and must be changed in production.
