## [v1.0.0] - 2026-07-09

### Added

- Initial public release
- Public API discovery workflow
- Horse and race data collection pipeline
- SQLite-based data warehouse
- Raw JSON preservation
- Data normalization pipeline
- Feature engineering workflow
- Machine learning prediction pipeline
- FastAPI read-only dashboard
- Dashboard screenshots
- Shadow-mode monitoring
- Betting simulation interface
- VPS deployment scripts
- Backup and cleanup automation
- MIT License
- Contributing guide
- Initial GitHub Issues for roadmap tracking

This release is intended for research and educational use.

Some dashboard metrics may appear as zero in local development mode until enough evaluated races and prediction results are collected.
